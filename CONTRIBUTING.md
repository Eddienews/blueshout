# Contributing

Thanks for helping improve Blueshout.

## Local Development

1. Create a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Copy `.env.example` to `.env` and set a local `SECRET_KEY`.
4. Use `SESSION_COOKIE_SECURE=0` for local HTTP development.
5. Run the app with `flask --app app run --host 127.0.0.1 --port 5000`.

## Checks

Before opening a pull request, run:

```bash
SECRET_KEY=test SESSION_COOKIE_SECURE=0 python -m py_compile app.py
```

If you change authentication, session handling, or TTS behavior, include a short note in the pull request explaining the security impact.

## Pull Requests

Keep changes focused. Avoid committing local secrets, generated bytecode, virtual environments, logs, or model files.
