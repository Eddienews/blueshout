# Code of Conduct

## Our Pledge

We want Blueshout to be a welcoming project for contributors with different backgrounds, experience levels, and perspectives. We expect participants to communicate with respect, assume good intent, and keep discussions focused on improving the project.

## Expected Behavior

- Be kind, direct, and constructive.
- Respect different viewpoints and constraints.
- Give actionable feedback when reviewing issues or pull requests.
- Avoid sharing private credentials, tokens, logs, or user data in public threads.

## Unacceptable Behavior

Harassment, personal attacks, discriminatory language, intimidation, doxxing, and repeated disruptive behavior are not acceptable in this project.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code of conduct. Maintainers may also limit participation when needed to protect the project and its community.

## Reporting

If you see behavior that violates this code, contact the maintainer privately. Please include relevant links or screenshots when possible.
